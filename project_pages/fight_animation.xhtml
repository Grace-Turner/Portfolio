Fighting Animation

The task for this project was to animate a run cycle and attack animation. I used a premade rigged human model and modelled weapons and sheaths to match in Autodesk Maya. For the run cycle, I used a reference video to plan the keyframes and animated it in Autodesk Maya. For the attack sequence, I used reference videos of myself and gathered from YouTube to plan the fight sequence and animated it in Autodesk Maya. My animation process is shown in the videos below.
