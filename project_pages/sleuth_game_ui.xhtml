Sleuth Game

User Interface Design

Above is the design for the game title screen. When an option is hovered over it will be lit up in red, as can be seen on "New Game". This is where the user will navigate to the game.

This is the design for the screen layout the user will see when playing as The Detective. Their character-specific ability can be seen in the bottom right-hand corner.

This is the design for the screen layout the user playing The Detective will see when they have activated the special ability Detect. The Detect icon is lit up red to indicate that it is activated, and a push pin with a string can be seen on the screen which can be used to select multiple clues to combine.

This is the design for the screen layout the user will see when playing as The Forensic Scientist. Their character-specific ability can be seen in the bottom right-hand corner.

This is the design for the screen layout the user playing The Forensic Scientist will see when they have activated the special ability Examine. The Examine icon is lit up red to indicate that it is activated, and a set of forceps can be seen on the screen which can be used to pick up scientific evidence.

This is the design for the screen layout the user will see when playing as The Photographer. Their character-specific ability can be seen in the bottom right-hand corner.

This is the design for the screen layout the user playing The Photographer will see when they have activated the special ability Record. The Record icon is lit up red to indicate that it is activated, and a camera focus box can be seen on the screen which can be used to take photos of evidence.

This is the design for the screen layout the user will see when playing as The Police Officer. Their character-specific ability can be seen in the bottom right-hand corner.

This is the design for the screen layout the user playing The Police Officer will see when they have activated the special ability Charge. The Charge icon is lit up red to indicate that it is activated, and cross hairs can be seen on the screen which can be used to aim and shoot the attacker.

This is the design for the screen layout the user will see when playing as The Psychologist. Their character-specific ability can be seen in the bottom right-hand corner.

This is the design for the screen layout the user playing The Psychologist will see when they have activated the special ability Analyse. The Analyse icon is lit up red to indicate that it is activated, and a focus light can be seen on the screen which makes additional information about suspects' relationships and whereabouts visible.

This is the design for the in-game map the player can use to help them navigate the mansion. The player's location is indicated by a black diamond.
